CSS Positioning Challenge

Practice static, relative, absolute, fixed, sticky and z-index. Edit styles.css to complete each task. No JavaScript required.

git

Tasks

  1. Center the blue box both vertically and horizontally inside the canvas (use absolute positioning + transform, or other positioning techniques).
  2. Make the yellow box stick to the top of the viewport when you scroll (use position: sticky).
  3. Fix the green box to the bottom-right corner of the viewport (use position: fixed).
  4. Offset the orange box 30px downward and 20px to the right relative to its normal position (use position: relative).
  5. Raise the purple box above the red box using z-index so the purple overlaps on top.
Hints
  • Use the class names in the canvas to target elements: .center-box, .sticky-box, .fixed-box, .relative-box, .red, .purple.
  • Open solution.css to check an example solution once you've tried the challenge.
